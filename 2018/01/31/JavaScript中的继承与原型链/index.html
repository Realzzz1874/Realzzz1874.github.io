JavaScript中的继承与原型链

Web JavaScript
 2018/01/31

个人觉得JavaScript中最难的地方就是闭包与原型链了吧。。。

JavaScript是基于对象的语言，所有的东西向上追溯都是对象，但是有趣（bug）的是它又不是真正的面向对象编程，因为它连个class都没有。（不要以为es6中引入了class关键字，那只是一个语法糖而已，JavaScript本身依然是基于原型的。。。）

OOP的核心思想就是：继承、封装、多态。

封装

类可以将属性（property）和方法（method）封装起来作为实例对象的引用。可是JavaScript中并没有实现类的概念。

JavaScript规定，每一个构造函数都有一个prototype属性，用来指向另一个对象。这个对象的所有属性和方法都会被构造函数的实例所继承。这就意味着，在封装一个类的时候就可以将那些不变的属性和方法直接定义在prototype对象上面。（部分代码示例参考这个网页）
