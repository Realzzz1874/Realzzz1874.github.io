ж·ұе…ҘдәҶи§ЈCommonJSжЁЎеқ—е®һзҺ°еҺҹзҗҶ

Web JavaScript
 2018/01/24

еңЁJavaScriptдёӯдҪҝз”ЁеҸҳйҮҸд№ҹйңҖиҰҒеҗҢе…¶е®ғзј–зЁӢиҜӯиЁҖдёҖж ·иҰҒжіЁж„ҸеҸҳйҮҸзҡ„дҪңз”ЁеҹҹпјҢеҰӮжһңa.jsе’Ңb.jsдёӯйғҪдҪҝз”ЁдәҶе…ЁеұҖеҸҳйҮҸxпјҢйӮЈд№Ҳе®ғд»¬е°Ҷдјҡдә§з”ҹеҶІзӘҒгҖӮ

JavaScriptиҜӯиЁҖжң¬иә«е№¶жІЎжңүдёҖз§ҚжЁЎеқ—жңәеҲ¶жқҘдҝқиҜҒдёҚеҗҢжЁЎеқ—д№Ӣй—ҙеҸҜд»ҘдҪҝз”ЁзӣёеҗҢзҡ„еҸҳйҮҸеҗҚгҖӮ

жЁЎеқ—зҡ„йҡ”зҰ»

JavaScriptжҳҜеҮҪж•°ејҸзј–зЁӢиҜӯиЁҖпјҢJavaScriptж”ҜжҢҒй—ӯеҢ…гҖӮз”Ёй—ӯеҢ…еҮҪж•°зҡ„ж–№ејҸе°ҶдёҖж®өJavaScriptд»Јз Ғе°ҒиЈ…иө·жқҘпјҢйӮЈд№Ҳиҝҷж®өд»Јз Ғдёӯзҡ„еҸҳйҮҸе°ҶеҸӘиғҪеңЁжң¬й—ӯеҢ…дёӯдә§з”ҹдҪңз”ЁпјҢйҮҮз”Ёиҝҷз§Қж–№жі•дҫҝеҪўиұЎең°е°ҶвҖңе…ЁеұҖеҸҳйҮҸвҖқеҸҳжҲҗдәҶй—ӯеҢ…дёӯзҡ„вҖңеұҖйғЁеҸҳйҮҸвҖқгҖӮ

зј–еҶҷдёҖдёӘjsж–Үд»¶е‘ҪеҗҚдёәa.jsпјҡ

Node.jsжЁЎеқ—еҢ–еҠ иҪҪд»ҘеҗҺдјҡе°ҶеҺҹжқҘзҡ„д»Јз ҒиҝӣиЎҢе°ҒиЈ…пјҡ

иҝҷдёҖжӯҘж“ҚдҪңд»ҘеҗҺпјҢеҺҹжқҘзҡ„е…ЁеұҖеҸҳйҮҸaзҺ°еңЁе°ұеҸҳжҲҗдәҶй—ӯеҢ…дёӯзҡ„еұҖйғЁеҸҳйҮҸгҖӮNodeеҲ©з”ЁJavaScriptй—ӯеҢ…зҡ„зү№жҖ§пјҢеҸҜд»ҘеҫҲе®№жҳ“зҡ„е°ұе®һзҺ°дәҶжЁЎеқ—дёҺжЁЎеқ—д№Ӣй—ҙзҡ„йҡ”зҰ»гҖӮ

жЁЎеқ—зҡ„иҫ“еҮә

еҮҶеӨҮдёҖдёӘжЁЎеқ—

еҮҶеӨҮдёҖдёӘmoduleеҜ№иұЎ

иҜҙжҳҺ

еҸҳйҮҸmoduleжҳҜNodeеңЁеҠ иҪҪjsж–Үд»¶д№ӢеүҚе°ұе·Із»ҸеҮҶеӨҮеҘҪзҡ„дёҖдёӘеҸҳйҮҸпјҢеңЁa.jsж–Үд»¶дёӯиғҪзӣҙжҺҘдҪҝз”Ёmoduleзҡ„еҺҹеӣ е°ұжҳҜеӣ дёәе®ғжң¬иә«е°ұжҳҜдёҖдёӘеҮҪж•°еҸӮж•°гҖӮ

йҖҡиҝҮеҸӮж•°жҠҠmoduleдј йҖ’з»ҷload()еҮҪж•°пјҢa.jsе°ұжҲҗеҠҹзҡ„жҠҠдёҖдёӘеҸҳйҮҸдј йҖ’з»ҷдәҶNodeзҡ„жү§иЎҢзҺҜеўғпјҢNodeдјҡжҠҠmoduleйӣҶдёӯдҝқеӯҳеҲ°дёҖдёӘең°ж–№гҖӮз”ұдәҺNodeдҝқеӯҳдәҶжүҖжңүзҡ„moduleпјҢеҪ“з”ЁrequireиҺ·еҸ–moduleзҡ„ж—¶еҖҷпјҢNodeжүҫеҲ°еҜ№еә”зҡ„moduleпјҢжҠҠиҝҷдёӘmoduleеҜ№еә”зҡ„exportsеҸҳйҮҸиҝ”еӣһгҖӮ
