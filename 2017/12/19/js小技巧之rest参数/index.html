js小技巧之rest参数

JavaScript js小技巧 tips
 2017/12/19

是什么？

rest 参数是形式为...的变量名，可以用来获取函数的多余参数，rest参数搭配的变量是一个数组，该变量将多余的参数放入数组中。

简单说：将多余的逗号分隔的参数序列转换为数组参数。（和扩展运算符spread相反）

怎么用

  • rest参数接收函数的多余参数，组成一个数组，要放在形参的最后

  • rest参数可以用来代替不确定参数个数的传递方式

add函数可以用来求任意个数参数的和。

rest参数和arguments对象的区别

  • rest参数只包含那些放在后面没有名字的参数，arguments对象包含了所有的参数

  • rest参数是一个array，arguments不是array。arguments为了使用数组的方法，必须使用Array.prototype.slice.call先将其转为数组。
