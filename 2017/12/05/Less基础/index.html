什么是Less

Less是一个CSS预处理器，能够定制，可管理和可重复使用网站的样式表。 Less是一个动态的样式表语言，扩展CSS的性能。

CSS预处理器是一种脚本语言，它扩展了CSS并将其编译成普通的CSS语法，以便它可以通过Web浏览器读取使用。它提供了像变量，函数，混合类型和操作，可以很方便的建立动态的CSS功能。

为什么要使用Less

  • Less创建更简洁，更容易，跨浏览器更友好
  • Less编译比其他的 CSS 预处理器更快
  • Less的模块化设计使得代码可读性提高
  • Less的变量设计使得项目更容易维护
