什么是组件？

组件 (Component) 可以扩展HTML元素，封装可以重用的代码。组件是自定义元素，Vue.js的编译器为它添加特殊的功能。

组件的作用域是相互孤立的，在子组件中无法直接引用父组件中的数据。

父子组件的关系为 props down，events up.

父组件通过prop给子组件下发数据，子组件通过事件给父组件传递消息。

动态props

与绑定到任何普通的HTML特性相类似，我们可以用v-bind来动态的将prop绑定到父组件的数据。每当父组件的数据变化的时候，该变化也会传导给子组件。

在 JavaScript 中对象和数组是引用类型，指向同一个内存空间，如果 prop 是一个对象或数组，在子组件内部改变它会影响父组件的状态。

子组件传递数据给父组件

在父子组件通信中，数据是单向传递的，子组件不能更改父组件的数据。如果子组件中需要更改父组件中的数据，可以先将数据保存下来然后再进行更改。更改完成以后如果子组件还想修改父组件的数据是不允许的，但是可以通过触发更改事件告知父组件，从而达到了子组件更改父组件的需求。
