是什么？

扩展运算符（spread）是由三个.组成，可以将一个数组转换成一个用逗号分隔的序列。

函数调用

这个例子简单说明了调用函数的时候可以将一个数组通过扩展运算符转换为一个参数序列。

函数调用的具体运用

在JavaScript中求一个数组的最大值我们可能需要写一个比如二分法的查找算法，而JavaScript的Math对象中已经封装了一个Math.max()的函数用来求最大值，不过缺点是所传的参数不能是一个数组，而扩展运算符完美地解决了这个弊端。

作用于实现了 Iterator 接口的对象

在处理DOM节点的时候，我们经常需要获取某一整个nodeList然后批量操作

这种方法可以简便地将nodeList转换为array进行操作。

扩展运算符内部调用的是数据结构的 Iterator 接口，因此只要具有 Iterator 接口的对象，都可以使用扩展运算符，比如 Map 和 Set 。

替代数组的apply方法

扩展运算符可以将一个数组转换成一个用逗号分隔的参数序列，所以在ES5中的apply方法就会被很好的替代。
